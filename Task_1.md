# Task 1 — Intermediate Airtable Skills

## New Hires Table Sample

You will be working with a table of **new hires** with the following fields:
1. **First Name**: The first name of the new hire.
2. **Last Name**: The last name of the new hire.
3. **Email Input**: The new hire's email address.
4. **Amount Spent on Equipment**: The amount spent by the new hire on equipment (numeric value).
5. **Created Date**: The date the new hire record was created.

### Your Objectives:
1. **Full Name Formula**:
   Create a formula to concatenate the First Name and Last Name fields into one Full Name field.

2. **Cleaned Email Formula**:
   Create a formula that cleans the Email Input field by removing extra spaces or special characters.

3. **Status Categorization Formula**:
   Create a formula field that categorizes new hires based on the amount spent on equipment:
   - **High Value**: Amount > 1000
   - **Medium Value**: Amount between 500 and 999
   - **Low Value**: Amount < 500

4. **Days Since Created Formula**:
   Create a formula field that calculates the number of days since the Created Date.

### Deliverables:
- Include screenshots of the table with the formulas applied.
- Provide a file `formulas.md` with the formula logic and explanations for each task.
